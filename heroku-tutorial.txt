// deploying on heroku
1. in package.json:
	a. The version of Node.js that will be used to run your application on Heroku, 
	should also be defined in your package.json file:
	"engines": {
    "node": "16.8.0"
	}
    
	b. "scripts": {
	"start": "node index.js"
	}
	
2. To determine how to start your app, Heroku first looks for a Procfile.
	so, create a procfile and in it type:
	web: npm start
	
3. make sure you have necessary packages before starting: 
   npm install //installs packages listed in package.json
   
4. heroku local web //Start your app locally

5. create .gitignore

6. git add .
7. git commit -m "Added a Procfile."
8. heroku login // Enter your Heroku credentials.
9. heroku create
10.git push heroku master/main
11. heroku open

notes:
